 RDQA 2026 - Dashboard Estratégico da Saúde Municipal

Sistema web desenvolvido para visualização estratégica de diretrizes, objetivos, metas e indicadores da Secretaria Municipal de Saúde de Itaúna/MG.

 Sobre o Projeto

O RDQA 2026 é um dashboard interativo criado para apresentar de forma moderna e visual os dados:

Relatório Detalhado do Quadrimestre Anterior (RDQA)
Diretrizes Municipais da Saúde
Objetivos Estratégicos
Indicadores de desempenho
Metas por gerência

O sistema foi desenvolvido com foco em:

✅ Visual moderno
✅ Responsividade
✅ Indicadores estratégicos
✅ Gestão pública em saúde
✅ Facilidade de leitura e apresentação institucional

 Tecnologias Utilizadas
HTML5
CSS3
JavaScript


 Funcionalidades
Dashboard estratégico em tempo real
Cards com indicadores principais
Gráficos interativos
Organização por diretrizes e objetivos
Visualização de metas por gerência
Layout responsivo
Interface moderna estilo Power BI/Web App

EXPLICAÇÃO DO DASHBOARD:

* TEMOS 68 OBJETIVOS MONITORADOS OU MELHOR DIZENDO ASSOCIADOS POR 70 INDICADORES;
* HÁ TAMBÉM 30 DIRETRIZES;
* O GRÁFICO HORIZONTAL (PRINCIPAIS METAS) DE CADA SECRETÁRIA, INCLUÍ BARRAS AZUIS DESTACANDO A QUANTIDADE DE METAS ALCANÇADAS POR CADA GERÊNCIA. 
JÁ AS DE CORES AMARELAS (METAS SECUNDÁRIAS) FORAM CITADAS A QUANTIDADE DE METAS NÃO BATIDAS / CONCLUÍDAS.
* A QUANTIDADE DE GERÊNCIA SEGUNDO OS DADOS OBTIDOS SÃO 15;
* O ÚLTIMO GRÁFICO SENDO MAIS ILUSTRADO EM CONCORDÂNCIA EXPLICATIVA DO GRÁFICO HORIZONTAL , ILUSTRADA A QUANTIDADE DE METAS BATIDAS POR GERÊNCIA.
DEIXANDO EM PALTA QUE 75% PRA CIMA FORAM ATINGIDAS, 50 Á 74% ESTÁ EM PROGRESSO OU MELHOR DIZENDO DENTRO DA MÉDIA POSSÍVEL, DE 50% PRA BAIXO FICARAM NEGATIVAS;
* METAS ALCANÇADAS ILUSTRADAS DE VERDE;
* AMARELO SÃO EM PROGRESSO , CRESCIMENTO;
* VERMELHO SALDO NEGATIVO;
